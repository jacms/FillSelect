Select C8 (v1.0.0)

SelectC8 is a library to make select more user friendly.

Example 1

            
    let fruits = [{id: 1,name: "Apple"},{id: 2,name: "Banana"},{id: 3,name: "Grapes"},{id: 4,name: "Watermelon"}];

    let example1 = new SelectC8({
        element: "example1",
        list: fruits,
    });

    example1.fill();
